# GenVideo 系统架构

## 概述

AI 视频生成系统，支持视频搬运和 AI 自动生成视频两大功能。

## 核心流程

### 剪映风格：输入文案 → AI生成素材 → 自动剪辑 → 成片

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         视频生成流程 (剪映模式)                              │
└─────────────────────────────────────────────────────────────────────────────┘

  ┌──────────────┐
  │  输入文案     │  文案/关键词/脚本/文章
  │  或关键词     │
  └──────┬───────┘
         │
         ▼
  ┌──────────────┐
  │ AI 扩展场景   │  根据时长生成 3-10 个场景描述
  │ 生成素材描述  │
  └──────┬───────┘
         │
         ▼
  ┌──────────────┐
  │ ComfyUI      │  调用 AI 生成多个视频素材片段
  │ 生成视频片段  │  (每个片段 3-5 秒)
  └──────┬───────┘
         │
         ▼
  ┌──────────────┐
  │  自动剪辑     │  ffmpeg 拼接 + 转场 + 字幕
  │  配乐合成     │  添加背景音乐
  └──────┬───────┘
         │
         ▼
  ┌──────────────┐
  │   成片       │  输出最终视频
  │              │  自动上传 OSS
  └──────────────┘
```

### 视频搬运流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         视频搬运流程                                          │
└─────────────────────────────────────────────────────────────────────────────┘

  ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
  │  平台下载     │ ──▶ │  AI 风格转换  │ ──▶ │  发布到       │
  │  (抖音/快手) │     │  (ComfyUI)   │     │  其他平台     │
  └──────────────┘     └──────────────┘     └──────────────┘
```

## 核心模块

### Backend (Go)

| 模块 | 路径 | 说明 |
|------|------|------|
| handler | `internal/handler/` | HTTP 处理器，API 入口 |
| service | `internal/service/` | 核心业务逻辑：视频生成、素材剪辑 |
| repository | `internal/repository/` | 数据访问层，MySQL/Redis 操作 |
| model | `internal/model/` | 数据模型定义 |
| config | `internal/config/` | 配置管理 |
| minimax | `pkg/minimax/` | Minimax API 客户端 |

### Frontend (React)

| 页面 | 说明 |
|------|------|
| VideoRepurposing | 视频生成（主功能）+ 视频搬运 |

## API 端点

### 视频生成（核心）

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/video/generate` | 输入文案生成视频 |

**请求体**:
```json
{
  "input": "未来城市夜景",
  "input_type": "keywords",
  "style": "documentary",
  "duration": 60,
  "aspect_ratio": "16:9",
  "music": "cinematic"
}
```

### 视频搬运

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/video/download` | 下载视频 |
| POST | `/api/v1/video/recreate` | 二次创作 |
| POST | `/api/v1/video/publish` | 发布视频 |

### 配置

| 方法 | 路径 | 说明 |
|------|------|------|
| GET/POST | `/api/v1/config/video-providers` | 视频平台配置 |
| GET/POST | `/api/v1/config/ai-providers` | AI 服务配置 |
| GET/POST | `/api/v1/config/oss` | OSS 配置 |

### 任务状态

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/task/:id` | 获取任务状态 |
| GET | `/api/v1/tasks` | 列出所有任务 |

## 技术实现细节

### 视频素材生成 (ComfyUI)

1. **场景扩展**: Minimax AI 将关键词扩展为多个场景描述
2. **素材生成**: ComfyUI 根据每个场景描述生成 3-5 秒视频片段
3. **拼接剪辑**: ffmpeg 拼接所有片段，添加转场效果
4. **音频处理**: 添加背景音乐，混合音频

### AI 提示词模板

```python
# 场景扩展
f"根据关键词生成{scene_count}个视频场景描述。关键词: {keywords}, 风格: {style}"

# 视频生成 (ComfyUI)
"Generate a video clip: {scene_description}, style: {style}, duration: 3-5s"
```

### 真人模拟操作

- 所有自动化操作都有人工延迟 (3-60 秒)
- 模拟人类输入速度
- 遵守平台风控规则

## 启动方式

```bash
# 构建并启动所有服务
docker-compose up -d --build

# 查看状态
docker-compose ps

# 查看日志
docker-compose logs -f

# 停止
docker-compose down
```
